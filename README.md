# Resume Builder Web App

A full-stack Resume Builder application that allows users to create, edit, and download professional resumes with multiple templates.

---

## 🚀 Features

* Create and edit resume sections (Education, Skills, Projects, Experience)
* Multiple resume templates
* Dynamic form handling
* User-friendly and responsive UI
* Download resume functionality (PDF support)

---

## 🛠️ Tech Stack

* **Frontend:** React, Tailwind CSS
* **Backend:** Node.js, Express
* **Database:** MongoDB (if used)

---

## 📂 Project Structure

* **client/** → Frontend React application
* **server/** → Backend APIs and logic

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/shivanibaravkar/Resume-builder-web-app.git
```

### 2. Install dependencies

#### For frontend

```bash
cd client
npm install
npm run dev
```

#### For backend

```bash
cd server
npm install
node server.js
```
